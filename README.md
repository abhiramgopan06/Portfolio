# 💼 Personal Portfolio Website

This is a modern and responsive **Personal Portfolio Website** built using HTML and CSS. It showcases my profile, skills, services, and projects in a structured and visually appealing way.

---

## 🌐 Sections Included

- 🏠 Home  
- 👤 About  
- 🛠️ Services  
- 💡 Skills  
- 📂 Portfolio  
- 📞 Contact  

---

## 📌 Features

- 📱 Responsive navigation bar with hamburger menu  
- 🎨 Clean and modern UI design  
- 👤 About Me section  
- 🛠️ Services with icons (Font Awesome)  
- 💡 Skills section (HTML, CSS, JavaScript, Git)  
- 📂 Portfolio section with project images  
- 📞 Contact form  
- 🧭 Smooth navigation using anchor links  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- Font Awesome
